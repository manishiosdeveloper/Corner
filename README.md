# Corner
Corner is a social app where users can share their favorite places and highlight them on a map. It allows others to discover new locations, react using emojis, comment, and bookmark posts. Whether itâ€™s your go-to coffee spot or a hidden gem in your neighborhood, Corner makes sharing and exploring favorite places fun and engaging.

# Key Features
Reactions: Emoji-based reactions to posts for quick and fun interactions.
Comment System: Users can comment on posts to ask questions, share thoughts, or give recommendations.
Bookmarks: Save posts to a personalized list for easy access later.
